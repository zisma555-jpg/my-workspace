# CPC-реклама (оплата за клик) на WB

_Источник: «CPC-реклама (оплата за клик) на Wildberries.pdf»_

---

## Главный принцип

CPC = платишь только за клики. Подходит для **мёртвых карточек**: плохие позиции, нет выручки, неликвид.

---

## CPC vs CPM: когда что использовать

| Ситуация | Выбор |
|---|---|
| Карточка мёртвая, позиции далеко | **CPC** |
| Нет истории продаж | **CPC** |
| Неликвид, залежавшийся товар | **CPC** |
| Нормальная карточка, хорошая позиция в органике | **CPM** |
| Есть история, отзывы, нормальный CTR | **CPM** |
| Нужна точечная работа с кластерами | **CPM** |

Почему не CPM для мёртвых карточек: далеко в выдаче → CPM дорог для подъёма; платите за показы, которые не конвертят.

---

## Настройка CPC

1. Создать кампанию с оплатой за клик
2. Начать с минимальной ставки
3. Постепенно повышать, если нужно больше трафика

---

## Рекомендуемая конфигурация для неликвида (Метод 4)

- **Поиск:** CPC — минимальные расходы, платите только за клики
- **Полки:** единая ставка — минимальная, дешёвый дополнительный трафик
- Вместе реабилитируют карточку

---

## Переход с CPC на CPM

Когда карточка ожила (есть заказы, отзывы, позиция подросла) → переходить на CPM:
- CPM эффективнее для нормальных карточек
- Точечная работа с кластерами
- Лучший контроль позиций, дешевле в пересчёте на заказ
